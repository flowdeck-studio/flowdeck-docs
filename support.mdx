## License & Activation

FlowDeck requires a license for full functionality.

- **Cursor Extension**: Activate via the FlowDeck panel in your editor
- **CLI**: Run `flowdeck license activate YOUR-KEY` or set `FLOWDECK_LICENSE_KEY` environment variable
